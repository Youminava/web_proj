# Web Proj - Drupal Support Website

Лендинг компании Drupal Coder - профессиональная поддержка и сопровождение сайтов на CMS Drupal.

## О проекте

Этот проект представляет собой одностраничный сайт (SPA) для компании, специализирующейся на поддержке и разработке сайтов на Drupal. Сайт включает информацию об услугах, тарифах, команде, отзывах клиентов и контактную форму.

## Технологии

- **Frontend**: React 19.1.1
- **Сборка**: Vite 7.2.4
- **Состояние**: Redux Toolkit
- **Роутинг**: React Router DOM
- **Стилизация**: CSS Modules
- **Развертывание**: GitHub Pages
- **Отправка форм**: Formcarry

### Предварительные требования

- Node.js (версия 16 или выше)
- npm или yarn

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   └── layout/         # Компоненты макета (Header, MobileMenu)
├── sections/           # Секции главной страницы
│   ├── hero/          # Герой-секция
│   ├── support/       # Услуги поддержки
│   ├── support-experience/  # Опыт и экспертиза
│   ├── tariffs/       # Тарифы
│   ├── cases/         # Кейсы работ
│   ├── team/          # Команда
│   ├── reviews/       # Отзывы
│   ├── partners/      # Партнеры
│   └── contact/       # Контакты и форма
├── data/              # Статические данные
├── assets/            # Изображения, шрифты, видео
├── styles/            # Глобальные стили
├── utils/             # Утилиты
├── app/               # Redux store, роутинг
└── pages/             # Страницы приложения
```

## Развертывание

Проект автоматически развертывается на GitHub Pages при пуше в ветку `main`.
